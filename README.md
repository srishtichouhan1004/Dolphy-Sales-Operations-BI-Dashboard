# PROJECT-Dolphy-India-Sales-Operations-Dashboard-Power-BI-
Business Intelligence dashboard featuring sales overview, regional analysis, product performance and operational insights with interactive filters and KPIs.

A 4-page dashboard built on a star-schema data model, analyzing company-wide sales, regional performance, product and operations.

🔹 Dashboard details:
• Home — navigation hub with global filters (Year, Quarter, Region, Category, Industry)
• Sales Overview — revenue, profit, orders, margin, target tracking
• Regional Analysis — region & branch-level revenue comparison
• Product & Operations — category performance, returns, inventory alerts
• Built using DAX measures, table relationships, and conditional formatting

🔹 Key insights:
• Sales missed target every year from 2021–25 — ~90% overall attainment
• North, South & West drive ~75% of revenue; Central lags at ~10%
• 2 return reasons (Wrong Item, Quality Issue) cause ~60% of all returns
• 49 products are currently below reorder level, concentrated in 3 branches

🔹 Tools Used : Microsoft PowerBI, Power Query Editor 


🔹 Conclusion:
Margins stayed healthy (~52%) across every region and category — meaning the real challenges are revenue and process-driven, not cost-driven, which makes them more directly fixable.
